# Jobs to Be Done (JTBD) Skill

Source: https://github.com/AliDujie/jtbd-knowledge-skill

Full JTBD toolkit merging 4 schools: Klement Progress Forces, Ulwick ODI, Wunker Jobs Atlas, Kalbach Job Stories.

## 13 Capabilities
1. Interview guide generation (Switch/ODI/Churn)
2. Survey design
3. Opportunity scoring
4. Priority matrix
5. Competitive analysis
6. Marketing copy generation
7. Growth & retention strategy
8. JTBD statement generation
9. Universal Job Map (8 stages)
10. Desired Outcome Statements
11. Job Stories generation
12. Adoption obstacle diagnosis
13. Jobs Atlas (7 dimensions)

## Triggers
Use when user mentions: user research, jobs to be done, JTBD, customer jobs, opportunity scoring, interview guide, product strategy, user motivation, why users switch

## Progress Forces Model
- Push (pain with current solution)
- Pull (attraction to new solution)
- Anxiety (fear of change)
- Inertia (habit/comfort)

Net momentum = (Push + Pull) - (Anxiety + Inertia)
