# Web Asset Generator

Source: https://github.com/alonw0/web-asset-generator

Generate professional web assets from logos or text.

## Trigger
Use when user mentions: favicon, icon, app icon, social image, Open Graph, OG image

## Asset Types
- Favicons: 16x16, 32x32, 96x96, favicon.ico
- App icons (PWA): 180x180, 192x192, 512x512
- Social/OG images: 1200x630 (Facebook/LinkedIn), 1200x675 (Twitter)
